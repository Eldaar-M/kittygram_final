# Описание проекта Kittygram

Kittygram — сервис для публикации данных о котиках. В публикациях можно выкладывать фотографии, выбирать цвет и возраст котика.
Публикации также можно редактировать и удалять.

# Технологии

- Python 3.x
- node.js 9.x.x
- frontend: React
- backend: Django
- nginx
- gunicorn
- Certbot
